# PasteMine 更新日志

## v1.0.1 (2025-11-22) - 权限优化版本 ✨

### 🎯 重要改进

#### ✅ 简化权限要求
- **移除了"输入监控"权限要求**
- **现在只需要一个权限：辅助功能**
- 更符合用户预期，与市面上其他剪贴板软件一致

#### 🔧 技术改进
- 从 NSEvent API 切换到 Carbon EventHotKey API
- 全局快捷键现在通过系统级 API 注册
- 更稳定、更可靠的快捷键响应

#### 📝 权限说明更新
- 更新了安装文档
- 明确说明只需要"辅助功能"权限
- 改进了首次安装的权限请求流程

### 🎮 功能保持不变

所有功能完全正常：
- ✅ 自动记录剪贴板内容
- ✅ ⌘⇧V 全局快捷键唤起窗口
- ✅ 点击历史记录自动粘贴
- ✅ 实时搜索和过滤
- ✅ 系统通知
- ✅ 托盘图标和菜单

### 💡 用户体验提升

**之前版本：**
- 需要授予 2 个权限：辅助功能 + 输入监控
- 可能让用户困惑

**现在版本：**
- 只需要 1 个权限：辅助功能
- 首次启动时系统自动弹出权限请求
- 授予权限后所有功能立即可用

## 如何更新

### 从 v1.0.0 升级到 v1.0.1

1. **卸载旧版本**
   ```bash
   cd /Users/lemonstyle/Documents/xcode_pj/pas
   ./uninstall.sh
   ```

2. **安装新版本**
   - 双击新的 DMG 文件
   - 拖拽安装到 Applications

3. **权限设置**
   - 如果之前设置过"输入监控"权限，可以移除了（不再需要）
   - 只需保留"辅助功能"权限
