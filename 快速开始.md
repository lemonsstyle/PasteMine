# PasteMine v1.1 - 快速开始指南

## 🚀 立即使用

### 安装
```bash
open PasteMine-1.1.dmg
```
将 PasteMine 拖到 Applications 文件夹。

---

## ⚠️ 重要：数据结构已更新

**本次更新修改了忽略应用的数据结构，旧版本的忽略列表将无法使用。**

### 如果你是从旧版本升级：

1. **打开设置** > 隐私 > 忽略应用
2. **删除所有旧的应用**
3. **重新添加应用**（现在更可靠了！）

---

## 📋 添加忽略应用（新方式）

### 步骤 1：打开设置
点击菜单栏图标 > 设置 > 隐私 > 忽略应用

### 步骤 2：选择应用
点击 **"选择应用"** 按钮

### 步骤 3：在文件选择器中找到应用
- 默认会打开 `/Applications` 目录
- 找到要忽略的应用（如 WeChat.app）
- 点击"选择"

### 步骤 4：确认添加成功
列表中会显示：
```
微信
com.tencent.xinWeChat
```
第一行是显示名称，第二行是 Bundle ID（技术标识）

---

## 🧪 测试过滤功能

### 测试忽略应用
1. 添加"微信"到忽略列表
2. 在微信中复制一些文字
3. 打开 PasteMine 主窗口
4. ✅ **确认没有新记录**

### 测试正常记录
1. 在浏览器/编辑器中复制内容
2. 打开 PasteMine 主窗口
3. ✅ **确认可以正常记录**

---

## 🔍 为什么这次修复更可靠？

### 之前的问题
- 微信文件名：WeChat.app
- 系统显示名：微信
- ❌ **名称不匹配，过滤失败**

### 现在的方案
- 使用 **Bundle Identifier**（应用唯一标识）
- 微信的 Bundle ID：`com.tencent.xinWeChat`
- ✅ **无论中英文环境，始终准确匹配**

## 🎯 其他隐私功能

### 忽略复制类型
默认已忽略：
- 密码类型（ConcealedType）
- 临时数据（TransientType）
- 1Password 等密码管理器的数据

可以添加更多类型。

### 退出时清空
- 开启后，手动退出应用时自动清除所有历史
- 适合特别注重隐私的场景
- ⚠️ 慎用：清空后无法恢复

## 🐛 遇到问题？

### 查看日志
1. 打开"控制台.app"
2. 搜索 "PasteMine"
3. 查看是否有错误信息

### 常见问题

**Q: 添加应用后还是会记录？**
A: 
1. 确认列表中显示了正确的应用名称和 Bundle ID
2. 重启 PasteMine
3. 查看控制台是否有 "⏭️ 已忽略应用" 的日志

**Q: 可以忽略多个应用吗？**
A: 可以！没有数量限制。

**Q: 如何查看某个应用的 Bundle ID？**
A: 
```bash
osascript -e 'id of app "应用名称"'
```
