# 📋 PasteMine 安装指南

## 🎯 欢迎使用 PasteMine！

PasteMine 是一个强大的 macOS 剪贴板历史管理工具，帮助您轻松管理和重用复制的内容。

---

## 💻 系统要求

- macOS 15.6 或更高版本
- 约 50 MB 可用磁盘空间

---

## 🚀 安装步骤

### 1️⃣ 安装应用

**将 PasteMine 图标拖拽到 Applications 文件夹**

就这么简单！

### 2️⃣ 首次启动

1. 打开"启动台"或"访达"
2. 找到并双击 **PasteMine**
3. 如果系统提示"无法打开"：
   - 右键点击 PasteMine
   - 选择"打开"
   - 点击"打开"确认

### 3️⃣ 授予权限（重要！）⚠️

应用启动后需要授予权限才能正常工作：

#### ✅ 通知权限
- 系统会自动弹出请求
- 点击"允许"

#### ✅ 辅助功能权限（必需）
**这是唯一需要手动设置的权限！**

1. 首次启动时，系统会自动弹出权限请求窗口
2. 点击"打开系统偏好设置"
3. 在"辅助功能"列表中找到 **PasteMine**
4. 点击左下角 🔒 解锁（需要输入密码）
5. 勾选 PasteMine ✅

**此权限用于：**
- ✅ 自动粘贴功能
- ✅ 全局快捷键 (⌘⇧V)

**💡 提示：** 
- 只需要授予"辅助功能"权限，不需要"输入监控"权限
- 授予权限后，应用的所有功能都能正常使用

---

## 🎮 开始使用

### 基本操作

1. **查看托盘图标**
   - 应用启动后，屏幕右上角会显示剪贴板图标 📋
   - 应用不会在 Dock 栏显示

2. **复制内容**
   - 在任意地方复制文字（⌘C）
   - 会收到系统通知
   - 内容自动保存

3. **打开历史窗口**
   - 按 `⌘⇧V`（Command + Shift + V）
   - 或点击托盘图标

4. **自动粘贴**
   - 打开文本编辑器
   - 按 `⌘⇧V` 打开窗口
   - 点击任意历史记录
   - 内容自动粘贴 ✨

### 快捷键

| 快捷键 | 功能 |
|--------|------|
| `⌘⇧V` | 显示/隐藏剪贴板历史窗口 |
| `⌘C` | 复制（系统默认，会被自动记录） |

### 其他功能

- **搜索记录**：在窗口顶部输入关键词
- **删除记录**：右键点击记录 → 删除
- **清空历史**：点击搜索栏右侧的垃圾桶图标
- **设置选项**：点击窗口右上角的齿轮图标
- **退出应用**：右键托盘图标 → 退出

## 🔒 隐私保护

- ✅ 所有数据本地存储，不上传云端
- ✅ 每次启动自动清空历史
- ✅ 不收集任何用户信息

---

## 🗑️ 卸载

如需卸载 PasteMine：

### 方法 1：使用卸载脚本（推荐）
1. 双击 DMG 中的 `卸载 PasteMine.command`
2. 按照提示操作

### 方法 2：手动卸载
1. 退出 PasteMine（右键托盘图标 → 退出）
2. 将 `/Applications/PasteMine.app` 移到废纸篓
3. 清空废纸篓

**注意：** 系统权限设置需要手动移除（如果需要）

## ❓ 常见问题

### Q: 快捷键 ⌘⇧V 无法唤起窗口？
**A:** 请确保已授予"辅助功能"权限。重启应用后生效。

### Q: 点击历史记录无法自动粘贴？
**A:** 请确保已授予"辅助功能"权限。

### Q: 托盘图标不显示？
**A:** 检查应用是否正在运行，或尝试重启应用。

### Q: 历史记录为空？
**A:** 这是正常的，应用每次启动会自动清空历史以保护隐私。
